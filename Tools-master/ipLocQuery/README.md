# Usage
#### Step1
```sh
install the dependencies:
pip install -r requirements.txt
```
#### Step2
```sh
run "python ipLocQuery.py -ip ipaddr"
run "python ipLocQuery.py -f ipaddrfile"
for example:
	python ipLocQuery.py -ip 202.108.22.5
	python ipLocQuery.py -f testIPs.txt
```
